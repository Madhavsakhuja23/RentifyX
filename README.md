# RentifyX 🏠🚗

RentifyX is a full-stack rental marketplace platform where users can rent and list properties and vehicles.  
The platform provides secure authentication, real-time messaging, online payments, wishlist management, and a modern responsive UI.

---

## 🌐 Live Demo

### Backend API
https://rentifyx-ff33.onrender.com

---

# ✨ Features

- 🔐 JWT Authentication
- 🔑 Google OAuth Login
- 🏠 Property Listings
- 🚗 Vehicle Listings
- ❤️ Wishlist System
- 💬 Real-time Chat using Socket.IO
- 🔔 Unread Message Notifications
- 💳 Razorpay Payment Gateway Integration
- 📱 Fully Responsive Design
- 📦 REST API Architecture
- ☁️ Cloud Deployment with Vercel & Render
